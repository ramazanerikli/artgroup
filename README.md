## Webpack Project

[Click](https://ramazanerikli.github.io/webpack-project/) for DEMO.

#### Technologies Used
- Webpack
- Bootstrap
- SASS
- FontAwesome
- Jquery
- Slick-carousel

#### Installation
Install the packages via npm or yarn.
```
npm install
```

#### Development
Run development page on **localhost:8080**

```
npm run dev
```
#### Build 
Build for production.

```
npm run build
```
